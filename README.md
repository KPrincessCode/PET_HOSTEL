# Title
Pet Hostel Management System


# Intodruction

The Pet Hostel Management System is a C#-based project developed in Visual Studio using Windows Forms App (.NET Framework) to provide a reliable and efficient solution for pet owners who need temporary care for their pets. Whether the owner is traveling for work or on vacation, this system ensures that pets receive the necessary care and accommodations in a well-managed hostel environment.

The system allows users to register, log in, and book pet stays by providing details like pet type, duration, and medical needs. It calculates costs, supports card or mobile payments, and issues a receipt with a unique token. Admins can manage users, costs, and bookings, making pet boarding more efficient and convenient.


# Features

### 1. Welcome Page  
- Displays **information about the developers and supervisor**.  
- Includes a **“Continue” button** that navigates users to the **Login Page**.  
- Provides a professional entry point to the system.

### 2. User Registration (Sign-up)  
- New users can register by providing the following details:  
  - **Email**  
  - **Username**  
  - **Date of Birth**  
  - **Password**  
- Registration details are securely saved for authentication and login.  

### 3. Login System  
- Two types of users can log in: **User** and **Admin**.  
- Both log in using **Username** and **Password** credentials.  

### 4. Forgot Password  
- Includes a **“Forgot Password”** option to reset passwords.  
- A **verification code** is sent to the registered email for secure password recovery.  

### 5. User Panel  
- Allows users to **book hostel rooms** for their pets by entering:  
  - Pet Type  
  - Pet Age  
  - Injection Status  
  - Medicine Needed  
  - Start Date and End Date  
- The system automatically **calculates the total amount** based on the above details.  
- The “Pay for Booking” button remains **disabled** until the total amount is checked and confirmed.  

### 6. Payment Methods  
- Users can select their **preferred payment method**  

### 7. Payment System  
- Users complete payment by entering **necessary payment details**.    

### 8. Print Receipt (PDF)  
- After successful payment, users can **print a PDF receipt** with a **unique token number** for their booking.  

### 9. Admin Panel  
- Admins can:  
  - **View user and admin data**  
  - **View and update pet cost information**  
  - **Add, update, or delete** users and admins  
  - Manage overall system data efficiently
